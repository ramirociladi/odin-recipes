# Recipe Website

In this project, I have built a basic recipe website using HTML. The website consists of a main index page with links to a few different recipes. Each recipe page includes an image of the dish, a description, a list of ingredients, and a list of steps for making the dish.

## Skills Demonstrated

- Creating and structuring HTML documents
- Linking to external resources (images)
- Using headings, paragraphs, and lists in HTML
- Creating unordered and ordered lists

This project has helped me practice and solidify my HTML skills. I have learned how to create well-structured HTML documents and how to include various types of content on a webpage. I am now able to link to external resources and use headings, paragraphs, and lists effectively in my HTML code.
