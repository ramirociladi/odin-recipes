# Odin Recipes

This project consists of a basic recipe website that was made as a part of the Foundations course of The Odin Project.

The goal was to practice the basics of HTML, such as:

- Using different HTML elements
- Using an HTML boilerplate
- Working with text
- Using lists
- Using links an images

Also, doing this project was a great way to getting familiar with the workflow of Git and GitHub using the command line.
